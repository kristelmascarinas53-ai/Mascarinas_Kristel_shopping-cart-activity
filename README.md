Description
This is a console-based Shopping Cart System written in C#. It allows users to select products, enter quantities, validate input, manage stock, and compute totals with discounts.
Features
- Product menu display
- Input validation (product and quantity)
- Stock checking
- Shopping cart system (no duplicates)
- Automatic stock deduction
- Receipt generation
- 10% discount for purchases ≥ ₱5000
- Updated stock display after checkout
